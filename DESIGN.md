## 전체 블로그 디자인
깔끔하고 심플한 블로그 디자인으로 만들었습니다.
흰색 배경에 푸른색, 보라색 계열로 포인트를 주고, 예쁜 디자인 보다는 가독성에 중심을 두었습니다.

### 1. 레이아웃
- 전체 최대 너비: 1200px
- 게시글 목록: PC는 3열, 태블릿은 2열, 모바일은 1열로 보이게
- 상세 페이지와 글쓰기 페이지는 가운데 정렬로 최대 720px

### 2. 헤더
- 상단에 고정되어 스크롤해도 항상 보이게
- 왼쪽에 블로그 이름, 오른쪽에 새 글 작성 버튼을 배치

### 3. 게시글 목록
- 썸네일 이미지, 태그, 제목, 요약, 날짜 표시
- 카드에 마우스를 올리면 효과 나타나게
- 썸네일이 없으면 기본 이미지가 나타나게

### 4. 검색
- 검색창에 제목을 입력하고 버튼을 누르면 검색

### 5. 태그 필터
- 태그 버튼을 클릭하면 해당 태그 글만 모아볼 수 있게

### 6. 상세 페이지
- 작성일과 수정일
- 본문 줄바꿈 그대로 유지

### 7. 글 작성 페이지
- 제목, 태그, 썸네일 URL, 요약, 본문 순서로 입력
- 제목과 본문은 필수 항목

### 8. 반응형
| 화면 | 기준 |
|---|---|
| 모바일 | 390px |
| 태블릿 | 768px |
| PC | 1200px |
